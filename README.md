## Requirment

For building and running the app you need:

- Node.js 18.12.1
- Yarn 1.22.19

## Script

```
$ yarn install
$ yarn start
```

## Stack

- react & react-dom 18.2.0
- react-router & react-router-dom 6.8.2
- styled-components 5.3.8
